
### 删除提问接口（POST）
http://118.24.83.20:3001/delete/question    
[入参示例]
```
{
	"question_id":"5a8198dadb21344ce0dbbbce"
}
```
[入参说明]  

question_id|
|-------|
问题的唯一id|

[出参示例]
```
{
    "code": "0",
    "msg": "success",
    "result": "no result"
}
```
[出参说明]

code | msg  | result | 说明
------- | ------- | -------|----|
0 | sucess | 无| 删除成功
00001 | 系统错误 | 空| 删除失败
00002 | 参数格式不正确 |空| 删除失败
20001 | 问题_id号不存在，请检查！ | 空|删除失败
40000 | 数据库连接失败|空|删除失败
