### 个人信息设置接口（POST）
http://118.24.83.20:3001/setting/user   
[入参示例]
```
"user_id":"5a86c7b49bca952c109b9671"
"age":"11"
"motto":"成功是失败之母"
"avatar":
```
[入参说明]

user_id | age|motto | img
------- | ------- | -------|-----    
用户的唯一id | 年龄（可为空）|座右铭（可为空）| 头像（可为空）

[出参示例]
```
{
    "code": "0",
    "msg": "success",
    "result": "no result"
}
```
[出参说明]

code | msg  | result | 说明
------- | ------- | -------|----|
0 | sucess | 无| 设置成功
00001 | 系统错误 | 空| 设置失败
00002 | 参数格式不正确 |空| 设置失败
20000 | 该用户未注册，无法找到！ | 空|设置失败
40000 | 数据库连接失败|空|设置失败
